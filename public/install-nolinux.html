Install on Mac or Windows

Unsupported operating systems

eLabFTW is a server software that runs in Docker containers on a GNU/Linux operating system. It is not to be installed on personal computers running Windows or MacOS.

Warning

eLabFTW MUST be installed on a GNU/Linux server, not a personal computer.

If you would like to run eLabFTW and don’t have the resources to deploy an instance yourself, have a look at the hosted offers by Deltablot.
